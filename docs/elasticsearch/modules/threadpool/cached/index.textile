h1. Cached ThreadPool Module

p. A cached thread pool is an unbounded thread pool that reuses previously constructed threads.

h2. Settings

p. Settings for the cached Thread Pool module should be set for within @threadpool.cached@. For example, the @keepAlive@ setting should be set using @threadpool.cached.keepAlive@.

|_. Setting |_. Description |
|keepAlive|A time based settings controlling how long an unused thread can be kept in the pool. Defaults to @60s@.|
|scheduledSize|The number of threads that will be used to execute scheduled tasks. Defaults to @20@.|
