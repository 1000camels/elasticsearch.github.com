h1. Fields

p. Allows to selectively load specific *stored* fields for each document represented by a search hit. Defaults to load the internal @_source@ field.

<pre class="prettyprint lang-js">
{
    "fields" : ["user", "postDate"],
    "query" : {
        "term" : { "user" : "kimchy" }
    }
}
</pre>

p. @*@ can be used to load all stored fields from the document.

p. Script fields can also be automatically detected and used as fields, so things like @_source.obj1.obj2@ can be used.
