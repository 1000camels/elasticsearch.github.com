h1. Admin Cluster API

p. This section describes administration APIs that revolve around the cluster. The following are the APIs:

|_. API |_. Description |
|"health":health|Retrieves the health of the cluster.|
|"state":state|Retrieves the state of the cluster.|
|"nodes_info":nodes_info|Retrieves the one or more (or all) nodes information.|
|"nodes_stats":nodes_stats|Retrieves the one or more (or all) nodes statistics.|
|"nodes_shutdown":nodes_shutdown|Shuts down one or more (or all) nodes.|
|"nodes_restart":nodes_restart|Restarts one or more (or all) nodes.|
