h1. Query Filter

p. Wraps any query to be used as a filter. Can be placed within queries that accept a filter.

<pre class="prettyprint lang-js">
{
    "constantScore" : {
        "filter" : {
            "query" : { 
                "queryString" : { 
                    "defaultField" : "content", 
                    "query" : "this AND that OR thus"
                }
            }
        }
    }
}
</pre>
