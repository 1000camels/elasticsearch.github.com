title: Exists Filter
cat: guide
---

p. Filters documents where a specific field has a value in them.

<pre class="prettyprint lang-js">
{
    "constant_score" : {
        "filter" : {
            "exists" : { "field" : "user" }
        }
    }
}
</pre>

h1. Caching

p. The result of the filter is always cached.
